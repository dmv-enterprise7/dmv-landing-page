# DMV Enterprise - Landing Page

Landing page institucional da **DMV Enterprise**, agência especializada em soluções de Inteligência Artificial para automação de processos em pequenos e médios negócios.

**[dmventerprise.com.br](https://dmventerprise.com.br)**

---

## Sobre o Projeto

Site desenvolvido para apresentar os serviços da DMV Enterprise: assistentes virtuais com IA, automação de processos, fluxos inteligentes de WhatsApp, web apps customizadas, qualificação automática de leads e gestão inteligente de agenda.

### Funcionalidades

- Design responsivo (desktop, tablet e mobile)
- Animações de scroll reveal otimizadas para performance
- Suporte bilíngue (PT-BR / EN)
- Integração direta com WhatsApp (botão flutuante + CTAs)
- Seção de estatísticas com contadores animados
- FAQ interativo com accordion
- SEO otimizado

## Stack

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica |
| CSS3 | Estilização, animações, responsividade |
| JavaScript (Vanilla) | i18n, scroll reveal, contadores, navbar |
| Google Fonts | Sora, DM Sans, JetBrains Mono |
| GitHub Pages | Hospedagem |
| Cloudflare | DNS + domínio customizado |

## Performance

- Imagens otimizadas e comprimidas (redução de ~98%)
- Lazy loading nas imagens abaixo do fold
- Zero dependências externas (sem frameworks, sem jQuery)
- Carregamento rápido em conexões móveis
