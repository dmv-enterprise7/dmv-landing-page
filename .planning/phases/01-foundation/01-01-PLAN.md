**Section 5: Starfield (INFRA-05)**

Create the starfield using TWO pseudo-elements on `.background-effects` — one for small stars (layer 1) and one for slightly larger stars (layer 2). Use `radial-gradient` to create 50-80 star positions total across both layers. Each gradient is a tiny white dot at a specific percentage position.

Layer 1 (`.background-effects::before`) — ~40 small stars (1px dots):
- Use `background-image` with approximately 40 `radial-gradient(1px 1px at X% Y%, rgba(255,255,255,OPACITY), transparent)` entries
- Opacity values vary between 0.3 and 0.7 for visual variety
- Positions spread across the full 0-100% range in both X and Y
- `background-size: 250px 250px` to tile across the viewport
- Set `opacity: 0.5` on the pseudo-element
- `z-index: 0`

Layer 2 (`.background-effects::after`) — ~25 slightly larger stars (1.5px dots):
- Use `background-image` with approximately 25 `radial-gradient(1.5px 1.5px at X% Y%, rgba(255,255,255,OPACITY), transparent)` entries
- Opacity values vary between 0.4 and 0.8
- Different positions from layer 1 to avoid overlap
- `background-size: 350px 350px` (different tile size creates non-repeating visual)
- Set `opacity: 0.35` on the pseudo-element
- `z-index: 0`

Both pseudo-elements must have:
```css
content: '';
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
```

Generate real varied percentage positions (do NOT use placeholder "..." — write out all 40 + 25 gradient values). Space the positions so they look random: avoid clustering and even grids.
